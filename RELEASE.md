# الإصدار 1.0.0 - UserBot المدمج الشامل

## 📅 تاريخ الإصدار
- **التاريخ:** أبريل 2024
- **الحالة:** مستقر (Stable)

## 🚀 ما الجديد في هذا الإصدار؟

### المميزات الرئيسية:
1. **دمج كامل للأكواد**: توحيد جميع الوظائف في ملف واحد `main.py` لسهولة الإدارة والتشغيل.
2. **أوامر الهوية والإدارة**:
   - `.readall`: مسح الإشعارات وقراءة كل الرسائل.
   - `.scan`: فحص الحساب والمجموعات.
   - `.profile`: عرض معلومات البروفايل.
   - `.نبذة`: تعيين أو عرض النبذة الشخصية.
   - `.afk` / `.unafk`: وضع الخروج والعودة.

3. **أدوات التجارة والعملاء**:
   - `.bill`: إنشاء فواتير مخصصة.
   - `.autoreply`: الرد التلقائي الذكي.
   - `.bcast`: بث رسائل جماعية للمجموعات أو القنوات.
   - `.scrape`: استخراج بيانات المستخدمين أو المجموعات.
   - `.sch`: جدولة المهام والرسائل.
   - `.log`: حفظ السجلات والنشاطات.

4. **أدوات المحتوى والذكاء**:
   - `.v2t`: تحويل الصوت إلى نص (Voice to Text).
   - `.str`: ترجمة النصوص فورياً باستخدام Deep Translator.

5. **أدوات المطورين**:
   - `.eval`: تنفيذ أكواد بايثون مباشرة (للمطورين فقط).
   - `.github`: عرض معلومات المستودع أو الروابط.

## 🛠️ المتطلبات التقنية
- **لغة البرمجة:** Python 3.8+
- **المكتبات الأساسية:**
  ```bash
  pip install pyrogram tgcrypto deep-translator SpeechRecognition pydub
  ```
- **متطلبات النظام:**
  - تثبيت `ffmpeg` لتحويل الملفات الصوتية (ضروري لأمر `.v2t`).

## 🔧 طريقة التشغيل
1. تأكد من تثبيت المتطلبات المذكورة أعلاه.
2. قم بتعديل متغيرات البيئة (مثل `API_ID`, `API_HASH`, `BOT_TOKEN`) في ملف `main.py` أو عبر متغيرات النظام.
3. شغل البوت بالأمر:
   ```bash
   python main.py
   ```

## ⚠️ ملاحظات هامة
- **الأمان**: لا تشارك ملف الجلسة (`*.session`) مع أي شخص.
- **الحدود**: انتبه لحدود الـ FloodWait في تليجرام لتجنب الحظر المؤقت.
- **الصلاحيات**: بعض الأوامر تتطلب صلاحيات مشرف في المجموعات أو القنوات.

## 🐛 إصلاحات الأخطاء المعروفة
- تم تحسين التعامل مع أخطاء الاتصال وإعادة المحاولة تلقائياً.
- تم ضبط دقة التعرف على الصوت في أمر `.v2t`.
- تم تحسين سرعة الرد في ميزة `.autoreply`.
