## 📋 نظرة عامة

نظام بوت تيليجرام متقدم مُصمم خصيصاً لإدارة أعمال **محمود أبو الفضل**، يجمع بين أدوات الإدارة الشخصية، التجارة الإلكترونية، التسويق الرقمي، وأدوات المطورين في كود واحد شامل.

---

## ✨ المميزات الرئيسية

### 🛡️ الهوية والإدارة الأساسية
- **`.readall`**: قراءة جميع الرسائل وتصفير الإشعارات دفعة واحدة.
- **`.scan`**: فحص الروابط والنصوص الواردة (بالرد على الرسالة).
- **`.profile [mode]`**: تبديل سريع بين الهويات المهنية:
  - `dev`: وضع المطور والسيبراني.
  - `mark`: وكالة AboElfadl Media.
  - `per`: الوضع الشخصي.
- **`.نبذة`**: عرض تلقائي لبطاقة تعريفية مهنية شاملة مع الروابط.
- **`.afk [السبب]`**: تفعيل وضع "غير متاح" مع تغيير الـ Bio والرد الآلي.
- **`.unafk`**: إيقاف وضع AFK وعرض تقرير بالرسائل الواردة أثناء الغياب.

### 💼 التجارة وإدارة العملاء
- **`.bill [العميل] [المنتج] [السعر]`**: توليد فاتورة إلكترونية منسقة مع تفاصيل الدفع (InstaPay).
- **`.autoreply on/off`**: تشغيل/إيقاف الرد الآلي الترويجي للعملاء الجدد.
- **`.bcast`**: بث رسالة واحدة لجميع المجموعات المشتركة فيها.
- **`.scrape`**: سحب بيانات أعضاء المجموعة وحفظها في ملف CSV.
- **`.sch [ثواني]`**: جدولة إرسال رسالة بعد وقت محدد.
- **`.log`**: أرشفة طلبات العملاء في الرسائل المحفوظة مع الطابع الزمني.

### 🤖 أدوات الذكاء والمحتوى
- **`.v2t`**: تفريغ المقاطع الصوتية (Voice Notes) إلى نص مكتوب (Speech-to-Text).
- **`.str`**: ترجمة صامتة للنصوص الأجنبية وإرسال النتيجة للمحفوظات دون إشعار المرسل.

### 💻 أدوات المطور
- **`.eval [الكود]`**: تنفيذ أكواد بايثون مباشرة داخل البوت (Live Terminal).
- **`.github [كلمة]`**: البحث عن مستودعات GitHub وعرض أفضل النتائج.

## 🚀 التثبيت والتشغيل

### 1. المتطلبات الأساسية
تأكد من تثبيت Python 3.8+ والمكتبات التالية:

```bash
pip install pyrogram deep-translator speechrecognition pydub requests tgcrypto
```

> **ملاحظة:** لأداة تفريغ الصوت (`.v2t`)، يجب تثبيت `FFmpeg` على النظام.

### 2. إعدادات التشغيل
قم بتعديل المتغيرات التالية في بداية الملف الرئيسي `main.py`:

```python
API_ID = 26170736              # استبدله بـ API ID الخاص بك
API_HASH = "your_api_hash"     # استبدله بـ API Hash الخاص بك
OWNER_ID = 1431886140          # معرف حساب المالك (أبو الفضل)
```

### 3. تشغيل النظام
```bash
python main.py
```

## 🔒 الأمان والخصوصية

- النظام يعمل بحساب المستخدم (Userbot) وليس بوت عادي.
- جميع الأوامر مقيدة بـ `OWNER_ID` لضمان عدم استخدام النظام من قبل آخرين.
- لا يتم تخزين البيانات الحساسة خارج جلسات Pyrogram المشفرة.
